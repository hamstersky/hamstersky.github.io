Data annotation platform

The goal of this project was to develop a data annotation platform aimed at reconstructing a dataset of traffic participants trajectories at one of the intersections in Copenhagen. The initial state in the platform is the output of an algorithm for reconstructing trajectories that I created as part of my research project in ITU. The platform provides an interface to label trajectories and make alternative trajectories.

The platform was built in Python and a popular data visualization framework - Bokeh.

Othello AI

The goal of the project was to implement an AI that can play the Othello game. It implements the Minimax algorithm and applies concepts such as alpha-beta pruning and heuristics for optimized performance. The project was part of the course in Introduction to Artificial Intelligence at ITU.

The implementation is done in Java.
